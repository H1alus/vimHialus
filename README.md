my personal neovim config - keeping it all in lua - surely it's not the best but it works for me just fine
### Required package 
the following packages are needed:  
- python3-pip
- python3-neovim
- python3-venv
- python3-virtualenv
- ripgrep
- npm
- latexmk (only if you use latex with miktex)
